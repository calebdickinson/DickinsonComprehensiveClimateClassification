This page contains detailed world and regional maps of each of the three categories that determine a climate in the Dickinson Comprehensive Climate Classification System. To generate your own maps, visit https://github.com/calebdickinson/DickinsonComprehensiveClimateClassification/tree/main and scroll down to the README for information. The most useful computer programs for mapmaking will be found in the ClimateFiles folder or the MiscellaneousFiles folder in the github repository.
